# Blog Platform with Auth & Roles

Node.js/Express + React + MongoDB full-stack blog.

## Features
- JWT Auth (login/register)
- Role-based access (user/admin)
- Posts (admin CRUD)
- Comments (user create, admin soft-delete, star ratings)
- Rich text editor
- Image uploads
- Responsive UI
